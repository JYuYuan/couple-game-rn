# カップルゲーム - カップル向けインタラクティブゲーム

[English](README.md) | [中文](README.zh.md) | [日本語](README.ja.md)

> ⚠️ **18+ コンテンツ警告**
> このアプリケーションは成人向けインタラクティブコンテンツを含み、成人カップルの使用を推奨します。一部のタスクやゲームコンテンツには親密なやり取りが含まれる場合があります。

カップル向けに設計されたマルチプレイヤーインタラクティブゲームアプリケーション。飛行棋、ラッキーホイール、地雷対戦、お絵描き伝言など、さまざまなゲームモードをサポート。**オンラインマルチプレイ**と**ローカルLAN**の2つのゲームプレイオプションを提供。楽しいゲームとタスクシステムを通じてカップル間のやり取りと理解を深めます。

## ✨ 機能

- 🎮 **複数のゲームモード**
  - **飛行棋**：クラシックな2人用ボードゲーム、特別なマスでタスクチャレンジ
  - **ラッキーホイール**：楽しい抽選ホイール、サプライズタスクやペナルティをランダムに抽選
  - **地雷対戦**：2人対戦の地雷ゲーム、スピードと運を競う
  - **お絵描き伝言**：創造的な絵描きゲーム、一人が描いてもう一人が当てる

- 🤖 **AI駆動機能**
  - **AI単語生成**：お絵描き伝言ゲーム用のインテリジェントな単語生成
  - **AIタスク作成**：パーソナライズされたカップルタスクを自動生成
  - **多言語サポート**：選択した言語にAIが適応
  - **カスタマイズ可能**：独自のAIサービスを設定（OpenAI互換API）

- 🌐 **柔軟な接続オプション**
  - **LANモード**：同じWiFi上でローカルにプレイ、インターネット不要、低遅延
  - **オンラインモード**：サーバー経由でリモートマルチプレイ、離れていても一緒にプレイ

- 🎯 **タスクシステム**
  - 豊富な内蔵カップルインタラクションタスクライブラリ
  - カスタムタスクセットをサポート、パーソナライズされたゲーム体験
  - さまざまなタスクタイプ：Q&A、アクション、親密なやり取りなど
  - 難易度段階システム（簡単/普通/難しい）
  - **AI生成タスク**：AIで無限のユニークなタスクを作成

- 🌍 **マルチプラットフォームサポート**
  - iOSネイティブアプリ
  - Androidネイティブアプリ
  - Webバージョン（PWAサポート）

- 🎨 **ユーザーフレンドリー**
  - クリーンで美しいインターフェースデザイン
  - スムーズなアニメーション効果
  - 完全な国際化サポート（English/中文/日本語）

## 🎲 ゲームモード

### 飛行棋モード ✈️
クラシックな2人用飛行棋ゲーム。サイコロを振って前進、最初にゴールに到達した者が勝利。
- **タスクマス**に着地：ランダムなタスクを引いてチャレンジを完了
- **相手の駒**に着地：相手をスタート地点に戻す
- LANおよびオンライン対戦をサポート

### ラッキーホイールモード 🎡
エキサイティングな抽選ホイールゲーム、運を試す瞬間！
- ホイールを回転、ポインターの停止位置がタスクを決定
- 複数のタスクタイプが混在、サプライズ満載
- 交代で操作、ゲームのインタラクションを増加

### 地雷対戦モード 💣
2人対戦の地雷ゲーム、スピードと戦略を競う。
- 同じ地雷原を共有、より多くの安全なマスを開くことを競争
- 地雷を踏むと減点、数字を開くと加点
- リアルタイム対戦、緊張感溢れる

### お絵描き伝言モード 🎨
想像力を解き放つ創造的な絵描きゲーム！
- **描画フェーズ**：1人のプレイヤーが与えられた単語を描く
- **推測フェーズ**：もう1人のプレイヤーが何を描いているか推測
- **AI単語生成**：難易度に基づいてインテリジェントに単語を生成
- **スコアリングシステム**：推測スピードと正確さに基づいてポイント獲得
- **複数の難易度**：簡単/普通/難しい、異なる制限時間
- カップルが一緒に笑い合うのに最適

### タスクシステム 📝
すべてのゲームを通じたコアシステム：
- **Q&Aタイプ**：理解を深める、お互いについての質問に答える
- **アクションタイプ**：楽しいチャレンジ、指定されたアクションを完了
- **親密タイプ**：感情を深める甘いやり取り（18+）
- **カスタムタイプ**：専用タスクを作成、パーソナライズされた体験
- **AI生成**：AIを使用して無限のカスタマイズされたタスクを作成

## 🤖 AI機能詳細

### AI単語生成（お絵描き伝言）
- 選択した難易度に基づいて絵描き用の単語を自動生成
- 言語設定に適応
- 各ゲームで豊富で多様な単語を保証
- AIが利用できない場合は内蔵単語ライブラリにフォールバック

### AIタスク生成
- AIでパーソナライズされたカップルタスクを生成
- タスクの難易度とタイプをカスタマイズ
- 無限のユニークなタスクを作成
- 複数の言語をサポート

### AI設定
設定で独自のAIサービスを設定：
- **API URL**：OpenAI互換のAPIエンドポイント
- **API Key**：あなたのAPIキー
- **モデル名**：使用するモデルを指定
- OpenAI、Azure OpenAI、または互換サービスで動作

## 💡 使用シーン

- 🏠 **自宅デート**：LANモードを使用、データ使用なし、低遅延、高体験
- 🌃 **遠距離恋愛**：オンラインモードでいつでもどこでも一緒にプレイ
- 🎉 **パーティーの余興**：友人の集まりでのインタラクティブミニゲーム
- 💑 **関係を深める**：タスクシステムでお互いをよりよく知る
- 🎁 **特別な日**：記念日、誕生日などの特別な機会のロマンチックなやり取り

> 💡 **推奨：LANモード**
>
> 同じWiFi上でLANモードを使用して楽しむ：
> - ⚡ ゼロ遅延のゲーム体験
> - 🔒 よりプライベートなデータ転送
> - 📱 モバイルデータの消費なし
> - 🌐 外部サーバーへの依存なし

## 🚀 クイックスタート

### ローカル開発

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/couple-game-rn.git
cd couple-game-rn

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm start

# iOS
npm run ios

# Android
npm run android

# Web
npm run web
```

## 🛠 技術スタック

### クライアント
- **フレームワーク**: React Native + Expo
- **ルーティング**: Expo Router
- **状態管理**: Zustand
- **国際化**: i18next
- **UI**: React Native Reanimated + Expo Vector Icons
- **ネットワーキング**: Socket.IO Client
- **AI統合**: OpenAI互換APIサポート

### サーバー
- **ランタイム**: Node.js
- **フレームワーク**: Express
- **リアルタイム通信**: Socket.IO
- **言語**: TypeScript
